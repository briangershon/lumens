Lumens Docs

Frameworkless web components with expressive visuals.

Lumens is a library of browser-native components for adding motion, atmosphere, and interactive discovery to host-owned layouts. This index is the visual entrypoint; each component also has its own dedicated install-and-API page.

  • Use this page to compare components and inspect previews.
  • Use the component docs page when you need install steps, attributes, properties, events, and a starter example.
