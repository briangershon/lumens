Open-Source Visualization Components

Lumens is a library of web components for visual discovery.

Lumens is a growing open-source library of web components that focus on visualizations: atmospheric headers, interactive ornament, and browser-native scenes that can make a page feel alive without asking the host app to adopt a framework.

  • Visual storytelling primitives for the open web
  • Interactive surfaces shaped by motion, glow, and depth
  • Installable packages with standalone browser bundles
Current Constellation

`lumens-starmap-banner` sets the tone: visible stars, constellations, and exotic unseen objects like black holes in an interactive field designed to layer into editorial headers.
